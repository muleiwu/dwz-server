# 木雷短网址 Logo 使用规范

## 资产

- `docs/brand/logo-symbol.svg`：图形标，适合 favicon、头像、侧边栏折叠态和工作区品牌图标。
- `docs/brand/logo-lockup.svg`：浅色背景横向版，适合 README、官网公开页、后台登录页。
- `docs/brand/logo-lockup-dark.svg`：深色背景横向版，适合深色导航、深色封面和反白场景。

## 设计意图

Logo 以“清爽流线短链”为主方向：浅蓝圆角底表达轻量、清爽和易部署；三条柔和入口流线代表长链接、多域名和活动流量；绿色中心节点代表短链生成、策略命中和成功跳转；右侧单一箭头代表压缩后的可追踪出口。图形不做具象符号和字形化表达，让品牌更适合公开页、后台登录页、README 和轻量私有化部署场景。

## 色彩

| 用途 | 色值 | 说明 |
| --- | --- | --- |
| 浅蓝底 | `#E0F2FE` | 图形标背景，表达清爽和轻量。 |
| 浅描边 | `#BAE6FD` | 保证白底可见，同时避免厚重外框。 |
| 流线蓝 | `#38BDF8` | 弧形入口流线，表达轻快接入。 |
| 主蓝 | `#0EA5E9` | 主路径和出口箭头，表达短链跳转。 |
| 状态绿 | `#22C55E` | 中心节点，表达策略命中和成功跳转。 |
| 深色文字 | `#0C4A6E` | 浅色背景上的主字标。 |
| 浅色文字 | `#FFFFFF` | 深色背景上的主字标。 |

## 使用尺寸

- 图形标最小建议尺寸：16px。
- 后台侧边栏建议尺寸：32px。
- 登录页、README、官网公开页建议使用横向版，高度 40px 至 64px。
- 图形标四周至少保留 12.5% 的安全留白，避免贴边裁切；不要额外加深色容器，否则会削弱轻快感。

## 接入建议

- 管理后台将 `preferences.logo.source` 指向 `/admin/logo-symbol.svg`，与构建后的 public 静态资源保持同源。
- 公开模板将 `templates/icons.html` 中的 `icon_logo` 替换为图形标的内联 SVG。
- favicon 由 `logo-symbol.svg` 导出 16px、32px、48px 图层后生成 `.ico`。
- 若用于深色导航或深色封面，使用 `logo-lockup-dark.svg`；浅色页面、README 和登录页使用 `logo-lockup.svg`。
